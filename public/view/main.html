About the Project

Why I Made This?

My goal for this project was to:

  • create an app that would increase the productivity of my church's mission department,
  • learn and grow as a developer by solving a real world problem and working with other people.

What does it do?

The problem that I tried to tackle was the lengthy time it took in making mission's teams. In order to create different teams, the mission's director would have to take various variables into consideration like personality, preferences, gender, and etc. My application tries to automate that process by making groups based on those variables, and allowing the director to easily make updates if she desired to make changes.

What did I learned from this experience?

Unfortunately, this project didn't pass the beta phrase and the director didn't use the application for 2016's mission trips because of timing. She was pressed for time and needed to create mission's team earlier than expected. Even though the project didn't go as planned, I'm grateful for the learning experience. I learned the process of creating an application for other people, and trying to solve a problem by using algorithms.
